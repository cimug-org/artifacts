IEC 61970 Series Standards

Standard Series Description
IEC 61970-3xx The IEC 61970-300 series of documents specify the common information model (CIM). The CIM is an abstract model that represents the objects in an electric utility enterprise typically needed to model the operational aspects of a utility.
IEC 61970-4xx IEC 61970-400 series of profile standards specify the semantic structure of data exchanged between components (or applications) and/or made publicly available data by a component. Thes documents describe the payloads that would be carried if applications are communicating via a messaging system, but the documents do not include the method of exchange, and therefore are applicable to a variety of exchange implementations.
IEC 61970-6xx IEC 61970-600-x series of profile standards specify the rules and profile definitions for the European Common Grid Model Exchange Standard (CGMES).
